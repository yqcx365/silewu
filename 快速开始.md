## 🎯 快速开始 - 3步部署

你的项目已经完全优化，可以立即部署！

### 第1步：创建GitHub仓库

访问 https://github.com/new 创建新仓库：
- 仓库名: `digital-immortality` 或 `silewu`  
- 类型: Public（公开）
- 不要勾选任何初始化选项

### 第2步：上传代码

在当前目录打开PowerShell，复制粘贴以下命令（记得替换YOUR_USERNAME和YOUR_REPO）：

```powershell
git init
git add .
git commit -m "🚀 数字永生计划 v2.2.0 - 完整优化版"
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

### 第3步：启用GitHub Pages

1. 访问你的仓库页面
2. 点击 `Settings`（设置）
3. 左侧菜单点击 `Pages`
4. Source部分选择：`main` 分支，`/ (root)` 目录
5. 点击 `Save`
6. 等待3分钟，刷新页面看到绿色提示
7. 点击链接访问你的网站！🎉

### 第4步：更新URL（重要）

部署成功后，编辑 `index.html` 第17和22行：
```html
<meta property="og:url" content="https://YOUR_USERNAME.github.io/YOUR_REPO/">
<meta property="twitter:url" content="https://YOUR_USERNAME.github.io/YOUR_REPO/">
```

然后提交更新：
```powershell
git add index.html
git commit -m "更新部署URL"
git push
```
