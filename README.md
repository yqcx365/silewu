# 🧠 数字永生计划 - "死了么？"

一个创意十足的签到激励Web应用，通过游戏化的方式提醒用户每日签到。

## ✨ 功能特点

### 核心功能
- **生命能量系统**：通过每日签到维持生命能量，不签到将导致能量衰减
- **连续签到奖励**：连续7天签到获得能量核心，30天获得特殊徽章
- **倒计时系统**：实时显示距离下次签到的剩余时间
- **死亡警告机制**：生命能量耗尽将触发死亡倒计时，可使用能量核心复活
- **签到日历**：可视化展示签到历史和漏签情况

### 辅助功能
- **邮件提醒**：设置邮箱接收每日签到提醒和低能量警告
- **成就系统**：解锁各种里程碑成就
- **数据管理**：导出个人数据，支持数据重置
- **矩阵特效背景**：科幻感十足的视觉效果

## 🎮 使用说明

### 快速开始
1. 直接用浏览器打开 `index.html` 文件
2. 点击"今日签到"按钮完成首次签到
3. 设置邮箱地址以接收提醒（可选）

### 游戏规则
- **生命能量**：初始100%，每日签到恢复15%
- **能量衰减**：每过24小时未签到，生命能量减少15%
- **连续签到**：超过1天不签到会中断连续记录
- **死亡机制**：生命能量归零后进入死亡倒计时
- **能量核心**：连续签到7天获得1个，可用于复活

### 成就系统
- 🌟 首次签到
- 🔥 坚持不懈（连续7天）
- 👑 永恒追求（连续30天）
- 🏆 百炼成钢（总计100天）
- 🔋 生命满溢（能量100%）
- ✉️ 生命连线（设置邮箱）

## 🛠️ 技术栈

- **纯前端实现**：HTML5 + CSS3 + Vanilla JavaScript
- **本地存储**：使用 localStorage 保存用户数据
- **响应式设计**：适配桌面端和移动端
- **动画效果**：CSS3 动画 + Canvas 特效

## 🔧 自定义配置

### 修改衰减速率
在 `script.js` 中修改：
```javascript
// 每24小时减少15%能量
const decay = daysSinceCheckin * 15;
```

### 修改签到奖励
```javascript
// 每次签到恢复15%
userData.lifeForce = Math.min(100, userData.lifeForce + 15);
```

### 修改核心奖励频率
```javascript
// 连续签到7天获得能量核心
if (userData.streakDays % 7 === 0) {
    userData.energyCores++;
}
```

## ⚠️ 注意事项

1. **数据存储**：所有数据保存在浏览器本地，清除浏览器数据会丢失记录
2. **邮件功能**：当前仅为模拟，实际使用需要配置后端邮件服务
3. **时区问题**：签到判断基于本地时间
4. **浏览器兼容**：建议使用现代浏览器（Chrome, Firefox, Edge）
